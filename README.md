# Arduino-Buzzer-Project
This project uses a buzzer and a 220-ohm resistor to limit the current, aiming to produce a 400Hz sound at specific intervals.
You can access the simulation created using Wokwi via this link: https://wokwi.com/projects/463725315563338753
